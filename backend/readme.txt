========= Use these commands to run the local server =========

* first, activate the virtual environment before you do anything, and make sure you're in backend * 

    .\venv\Scripts\activate

* next, run the server over localhost *

    uvicorn app.main:app --reload


========== Running the frontend ============

* first, make sure you are in the frontend dir in the frontend terminal, then run *

    npm run dev
